# My Horario U 📚

**My Horario U** es una solución integral para estudiantes universitarios que permite organizar la vida académica, sincronizar datos en la nube y colaborar con compañeros compartiendo horarios de clase.

---

# ✨ Características

* **🔥 Sincronización en la Nube**: Tus datos siempre seguros y accesibles desde cualquier dispositivo gracias a Firebase.
* **📱 Social & QR**: Comparte tu perfil mediante códigos QR y agrega a tus amigos.
* **🤝 Comparador de Horarios**: Mira el horario de tus amigos lado a lado y detecta huecos libres comunes para estudiar.
* **📥 Importación Social**: Copia materias y horarios de tus amigos a tu propia agenda con un solo toque.
* **📚 Gestión de Materias y Tareas**: Organización completa de carga académica y actividades pendientes.
* **🕒 Temporizador en Vivo**: Visualiza cuánto falta para terminar tu clase actual directamente en el Dashboard.
* **📝 Notas y Evaluaciones**: Control de cortes académicos y cálculo automático de promedios.
* **🌙 Modo Oscuro Nativo**: Interfaz moderna adaptable a tus preferencias visuales.

---

# 🛠 Tecnologías utilizadas

* **Flutter & Dart**
* **Firebase** (Authentication & Firestore)
* **SQLite** (Persistencia local offline)
* **Mobile Scanner** (Escaneo de QR)

---

# 🚀 Instalación

1. Clonar el repositorio
```bash
git clone https://github.com/tuusuario/my-horario-u.git
```
2. Instalar dependencias
```bash
flutter pub get
```
3. Configurar Firebase
Ejecutar `flutterfire configure` para vincular tu archivo `google-services.json`.

4. Ejecutar la aplicación
```bash
flutter run
```
