# My Horario U 📚

**My Horario U** es una aplicación móvil diseñada para ayudar a estudiantes universitarios a organizar sus materias, horarios y actividades académicas de forma sencilla desde el celular.

La aplicación permite visualizar el horario semanal, registrar materias y gestionar información académica en un solo lugar.

---

# ✨ Características

* 📚 Gestión de materias
* 🕒 Organización del horario semanal
* 📝 Registro de notas y evaluaciones
* 📊 Visualización de promedios
* 📅 Panel principal con resumen académico
* 🌙 Modo claro y modo oscuro
* 🎨 Interfaz simple enfocada en estudiantes

# 🚀 Instalación

1. Clonar el repositorio

```bash
git clone https://github.com/tuusuario/my-horario-u.git
```

2. Entrar al proyecto

```bash
cd my-horario-u
```

3. Instalar dependencias

```bash
flutter pub get
```

4. Ejecutar la aplicación

```bash
flutter run
```

---

# 📦 Build de la aplicación

Para generar el APK:

```bash
flutter build apk --release
```

El archivo generado estará en:

```
build/app/outputs/flutter-apk/app-release.apk
```
